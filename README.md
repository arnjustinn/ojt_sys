OJT Console

A modern dark themed internship log system built with PHP and MySQL. This application tracks daily tasks, work hours, and progress toward a 486 hour internship goal.

Features

User authentication for private log management.

Dashboard with session metrics.

Analytics page showing progress percentage and remaining hours.

Absence tracking with reason logging.

Fully responsive dark mode interface.

Tech Stack

Frontend: Tailwind CSS, Vanilla JavaScript.

Backend: PHP (PDO).

Database: MySQL.

Installation

Clone this repository to your XAMPP htdocs folder.

Import the provided database.sql file into phpMyAdmin.

Configure your database credentials in config/db.php.

Access the application at http://localhost/ojt_sys.

Usage

Register a new identity.

Log in to access your private dashboard.

Deploy entries daily to track your progress.

View analytics to monitor your 486 hour goal.
