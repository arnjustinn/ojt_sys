# **OJT Console**

A modern dark themed internship log system built with PHP and MySQL. This application tracks daily tasks, work hours, and progress toward a 486 hour internship goal.

## **Features**

* User authentication for private log management.  
* Dashboard with session metrics.  
* Analytics page showing progress percentage and remaining hours.  
* Absence tracking with reason logging.  
* Fully responsive dark mode interface.

## **Tech Stack**

* Frontend: Tailwind CSS, Vanilla JavaScript.  
* Backend: PHP (PDO).  
* Database: MySQL.

## **Installation**

1. Clone this repository to your XAMPP htdocs folder.  
2. Import the provided database.sql file into phpMyAdmin.  
3. Configure your database credentials in config/db.php.  
4. Access the application at http://localhost/ojt\_sys.

## **Usage**

1. Register a new identity.  
2. Log in to access your private dashboard.  
3. Deploy entries daily to track your progress.  
4. View analytics to monitor your 486 hour goal.
